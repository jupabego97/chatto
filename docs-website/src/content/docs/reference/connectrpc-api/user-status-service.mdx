## Services

<a id="chatto-api-v1-UserStatusService"></a>

### UserStatusService

Manages the current user's custom profile status.

<a id="chatto-api-v1-UserStatusService-SetCustomStatus"></a>

#### SetCustomStatus

Sets or replaces the current user's custom status. Emoji and text are
required, and expires_at must be omitted or in the future.

```http
POST /api/connect/chatto.api.v1.UserStatusService/SetCustomStatus
```

<a id="chatto-api-v1-SetCustomStatusRequest"></a>

##### Input: SetCustomStatusRequest

Request to set or replace the current user's custom status.

| Field | Type | Description |
| --- | --- | --- |
| `emoji` | `string` | Short emoji marker displayed with the status. |
| `text` | `string` | User-written status text. |
| `expires_at` | `google.protobuf.Timestamp` | Optional future time after which clients should stop showing the status. |


<a id="chatto-api-v1-SetCustomStatusResponse"></a>

##### Result: SetCustomStatusResponse

Result of setting the current user's custom status.

| Field | Type | Description |
| --- | --- | --- |
| `status` | `CustomUserStatus` | Stored custom status after validation and normalization. |


<a id="chatto-api-v1-UserStatusService-ClearCustomStatus"></a>

#### ClearCustomStatus

Clears the current user's custom status. The call is idempotent and returns
the resulting empty status state.

```http
POST /api/connect/chatto.api.v1.UserStatusService/ClearCustomStatus
```

<a id="chatto-api-v1-ClearCustomStatusRequest"></a>

##### Input: ClearCustomStatusRequest

Request to clear the current user's custom status.


<a id="chatto-api-v1-ClearCustomStatusResponse"></a>

##### Result: ClearCustomStatusResponse

Result of clearing the current user's custom status.

| Field | Type | Description |
| --- | --- | --- |
| `status` | `CustomUserStatus` | Current custom status after the operation. Usually absent after a clear. |
