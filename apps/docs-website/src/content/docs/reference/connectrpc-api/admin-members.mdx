## AdminMemberService

Server-admin user identity and cooldown commands.

<a id="chatto-admin-v1-AdminMemberService-ListMembers"></a>

### ListMembers

Lists server members for the admin members screen. Requires
admin.view-users.

```http
POST /api/connect/chatto.admin.v1.AdminMemberService/ListMembers
```

<a id="chatto-admin-v1-ListMembersRequest"></a>

#### Input: ListMembersRequest

Request server-admin member rows.

| Field | Type | Description |
| --- | --- | --- |
| `search` | `string` | Case-insensitive login/display-name search term. Empty disables search. |
| `page` | `chatto.api.v1.PageRequest` | Page request. Defaults to 20 results when absent or limit is zero. |


<a id="chatto-admin-v1-ListMembersResponse"></a>

#### Result: ListMembersResponse

Server-admin member rows plus role summaries.

| Field | Type | Description |
| --- | --- | --- |
| `users` | repeated [`AdminMember`](/reference/connectrpc-api/types/#chatto-admin-v1-AdminMember) | Matching users. |
| `roles` | `repeated chatto.api.v1.Role` | Public roles for display-name lookup. |
| `page` | `chatto.api.v1.PageInfo` | Page metadata. |


<a id="chatto-admin-v1-AdminMemberService-GetMember"></a>

### GetMember

Gets one server member plus role/permission metadata for admin details.
Requires admin.view-users. Returns NOT_FOUND when the user does not exist.

```http
POST /api/connect/chatto.admin.v1.AdminMemberService/GetMember
```

<a id="chatto-admin-v1-GetMemberRequest"></a>

#### Input: GetMemberRequest

Request one server-admin member detail record.

| Field | Type | Description |
| --- | --- | --- |
| `user_id` | `string` | Target user ID. |
| `login` | `string` | Target login. Provide either user_id or login. |


<a id="chatto-admin-v1-GetMemberResponse"></a>

#### Result: GetMemberResponse

Server-admin member detail payload.

| Field | Type | Description |
| --- | --- | --- |
| `member` | [`AdminMember`](/reference/connectrpc-api/types/#chatto-admin-v1-AdminMember) | Member record. |
| `roles` | repeated [`AdminRole`](/reference/connectrpc-api/types/#chatto-admin-v1-AdminRole) | Roles for assignment UI. |
| `available_permissions` | `repeated string` | Permissions available for per-user overrides. |
