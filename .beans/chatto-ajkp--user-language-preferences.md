Allow users to configure their language preferences in user settings:

- **Primary language**: The user's main/native language
- **Other spoken languages**: Additional languages the user can communicate in

These preferences will be displayed on user profiles and can be used for:
- Matching users who speak the same languages
- Informing other users about communication preferences
- Future: Translation features or language-based room recommendations
